# Challenge 01 – Reverse a Linked List
**Difficulty:** Easy  
**Date:** 2025-05-01
___
### Problem Statement  
Given the head of a singly linked list, reverse the list and return the new head.

### Example  

    Input : 1 → 2 → 3 → 4 → null  
    Output: 4 → 3 → 2 → 1 → null

### Constraints  
* 0 ≤ nodes ≤ 5000  
* –5000 ≤ Node.val ≤ 5000
